createChatThread

  • createChatThread(name: string, msgId: string, parentId: string): Promise<ChatMessageThread>
    • Defined in ChatManager.ts:1869

    Creates a message thread.

    Each member of the group where the thread belongs can call this method.

    Upon the creation of a message thread, the following will occur:

    • In a single-device login scenario, each member of the group to which the message thread belongs will receive the {@link ChatMessageEventListener#onChatMessageThreadCreated} callback. You can listen for message thread events by setting ChatMessageEventListener.

    • In a multi-device login scenario, the devices will receive the {@link ChatMultiDeviceEventListener#onThreadEvent} callback. You can listen for message thread events by setting ChatMultiDeviceEventListener.

    throws

    A description of the exception. See ChatError.

    Parameters

    • name: string

      The name of the new message thread. It can contain a maximum of 64 characters.

    • msgId: string

      The ID of the parent message.

    • parentId: string

      The parent ID, which is the group ID.

    Returns Promise<ChatMessageThread>

    If success, the new message thread object is returned; otherwise, an exception will be thrown.
